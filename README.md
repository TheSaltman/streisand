streisand
=========

A private BitTorrent tracker backend written in python, django, and redis

To get started:
---------------

- download the Vagrant box using the torrent file in this repo
- place the box in the project root (next to Vagrantfile)
- `vagrant up`
- `vagrant ssh`
- `runserver`
